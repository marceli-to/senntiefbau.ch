Prerequisites
-------------
- PHP >= 8.2
- Composer
- Node.js & npm


Setup
-----
1. Install PHP dependencies:
   composer install

2. Create environment file:
   cp .env.example .env
   php artisan key:generate

3. Install JS dependencies:
   npm install

Or run everything at once:
   composer setup


Development
-----------
Start the dev server (Laravel + Vite + queue + logs):
   composer dev

This runs concurrently:
   - php artisan serve       (Laravel server)
   - npm run dev             (Vite dev server with HMR)
   - php artisan queue:listen
   - php artisan pail        (log viewer)


Production Build
----------------
Build front-end assets (CSS/JS):
   npm run build

Output goes to public/build/ with hashed filenames.


Static Export (Deploy)
----------------------
The site is exported to static HTML using spatie/laravel-export:
   php artisan export

This generates the dist/ folder with:
   - index.html
   - build/          (hashed CSS/JS)
   - img/            (images)
   - .htaccess       (rewrites, gzip, caching, security headers)

Upload the contents of dist/ to the web server.


Full Deploy Steps
-----------------
1. npm run build
2. php artisan export
3. Upload dist/ to server
